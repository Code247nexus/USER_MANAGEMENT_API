# User Management API 🚀

A RESTful User Management API built with FastAPI and Python.
This project is part of an ongoing series aimed at building
a production-ready backend system — progressively.

## 🔧 Tech Stack
- Python
- FastAPI
- Pydantic
- JSON (V1 storage)

## ✅ Features (Version 1)
- Full CRUD operations (Create, Read, Update, Delete)
- Input validation using Pydantic
- Filter users by attributes (e.g. active status)
- Sort users by age or creation date
- Structured error handling
- JSON-based data storage

## 📈 Version Roadmap
| Version | Focus | Status |
|---------|-------|--------|
| V1 | Core CRUD + Validation + JSON Storage | ✅ Done |
| V2 | MySQL Database Integration | 🔄 In Progress |

## ⚙️ How to Run
1. Clone the repository
git clone https://github.com/yourusername/user-management-api
2. Install dependencies
pip install -r requirements.txt
3. Run the server
uvicorn main:app --reload
4. Open API docs
http://localhost:8000/docs
